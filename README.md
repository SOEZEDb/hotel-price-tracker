Hotel Price Tracker
Tracks the cheapest hotel prices for any city or town worldwide and stores results in a spreadsheet. Automatically updates when a cheaper price is found.
Tech Stack: Python, Requests, Sheety API, Booking.com RapidAPI, schedule library
Features:
Dynamic check-in/check-out dates
CSV export
24hr auto-scheduling
Error handling
Auto-updates spreadsheet when cheaper price is found
How to Run:
Clone the repo, add .env with your API keys, run main.py
Required Environment Variables:
RAPIDAPI_KEY
SHEETY_ENDPOINT
